## Installation

1. Install Python 3.8+.
2. Install dependencies: `pip install -r requirements.txt`
3. For quantum: `pip install qiskit`
4. For image processing: `pip install opencv-python`
5. For bioinformatics: `pip install biopython`
6. For TensorFlow: `pip install tensorflow` (optional, large download, may not be available for all Python versions)

## Usage

- Run `python main.py` for the integrated demo.
- Use `python automate.py format` to format code.
- Open `ocular_singularity.ipynb` in Jupyter for mathematical content.
